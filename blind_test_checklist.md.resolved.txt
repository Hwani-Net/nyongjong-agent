# 🧪 뇽죵이Agent MCP → 스킬 대체 블라인드 테스트 체크리스트

> **용도**: 에이전트에게 새 프로젝트를 지시한 뒤, 작업 완료 후 이 체크리스트를 보여주고 자가 평가하게 한다.
> **핵심 원칙**: 이 문서는 테스트 전까지 에이전트가 볼 수 없는 곳에 보관한다.
> **테스트 방법**: 새 대화 → "○○ 앱 만들어줘" → 완료 후 → 이 체크리스트 붙여넣기 → "이 기준으로 자가 평가해"

## 📋 테스트 시나리오 (추천)

**지시문 예시**: "소상공인용 AI 서류봇 만들어줘. 견적서 자동 생성이 핵심이야."
- 새 프로젝트 (기존 코드 없음)
- medium~high 복잡도
- 웹 앱 (디자인 + 코딩 모두 필요)
