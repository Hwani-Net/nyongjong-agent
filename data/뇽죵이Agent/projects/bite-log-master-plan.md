## 6. 가드레일 원칙

| 원칙 | 구현 |
|------|------|
| 날짜/예약 = DB only | LLM이 날짜를 추론/생성하지 않음 |
| 신뢰도 점수 | LLM 결과에 정규식 기반 신뢰도 부여 |
| 출처 표시 | 모든 AI 답변에 원본 링크 명시 |
| 면책 문구 | "AI 참고용, 실제와 다를 수 있음" |
| Circuit Breaker | LLM 장애 시 DB 검색 폴백 |
| Rate Limiting | 유저당 일 50건 제한 |
