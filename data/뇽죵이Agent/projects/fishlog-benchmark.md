# FishLog 경쟁사 벤치마크 결과 요약

## 조사일: 2026-03-01
## 범위: 한국 5개 + 글로벌 9개 앱

## 핵심 발견
- 한국 1위 직접 경쟁: **피싱노트** (17만 DL) — "사진 1장 → 위치/날씨/물때 자동 저장"이 킬러피처
- 글로벌 1위: **Fishbrain** (1500만 유저) — AI BiteTime, Fish-Verify, $800K/월 매출
- 사용자 최대 불만: 핵심 기능 유료화 (Fishbrain 월 $13 구독)
- 사용자 최대 요청: 사진 자동 기록, AI 어종 인식, 입질 예측

## FishLog 강점 (차별화)
1. 핵심 기능 100% 무료
2. 오프라인 퍼스트 (IndexedDB 큐)
3. 풍부한 통계/분석 (차트, 어종비율, TOP5)
4. 캘린더 뷰 + 배지 시스템 (경쟁사에 없음)
5. Firebase 클라우드 동기화

## FishLog 약점 (P0 보완 필요)
1. ❌ GPS 위치 자동 감지 — 피싱노트의 핵심과 동등해야 함
2. ❌ 날씨 자동 기록 — 날씨 API 연동 필요
3. ❌ 물때 정보 — 한국 바다낚시 필수
4. ❌ 지도 뷰 — 포인트 시각화

## 우선순위
- P0: GPS 자동 위치 + 날씨 API
- P1: 물때 연동 + 지도 뷰
- P2: SNS 공유 카드 + AI 어종 인식
